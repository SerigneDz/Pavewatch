PaveWatch Setup

Two quick steps before your first test drive. Edit the CONFIG block at the top of this file.

STEP 1 · GOOGLE MAPS API KEY

Go to console.cloud.google.com → Create project → Enable Maps JavaScript API → Create API key → paste into CONFIG.GOOGLE_MAPS_KEY.

Under API restrictions, add your GitHub Pages domain (e.g. yourname.github.io). Free tier covers ~28,000 map loads/month.

STEP 2 · FIREBASE PROJECT

Go to console.firebase.google.com → Add project → Add Web app → copy the config object into CONFIG.FIREBASE.

Then: Firestore Database → Create database → Start in test mode. All your drive data saves automatically.
