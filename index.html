PaveWatch Setup

Edit the CONFIG block at the top of this file with your API keys, then tap below.

STEP 1 · GOOGLE MAPS API KEY

console.cloud.google.com → New project → Enable Maps JavaScript API → Credentials → Create API key → add your domain under restrictions → paste into CONFIG.GOOGLE_MAPS_KEY.

STEP 2 · FIREBASE

console.firebase.google.com → Add project → Add Web app → copy the 6-value config object into CONFIG.FIREBASE.

Firestore → Create database → Test mode → pick northamerica-northeast2 (Montreal).
